💾 Backup & Restore — all data
💾 Backup saves everything — tasks, gaps, budget, shopping, gifts, meals, Haru, Jimson, Home — into one pauline_backup_[date].json file.
📂 Restore loads that file back on any device, any browser, anytime — even after clearing cache or opening a new version.
The green 💾 Backup button at the bottom-right of every screen does the same thing.
📂 Restore from backup file
Select your pauline_backup_[date].json file. All your entries (tasks, budget, gaps, gifts, etc.) will be restored instantly.
⏰ Recommended habit: Tap 💾 Backup every Friday before closing the app.
📁 Where to save: Google Drive → Pauline's Hub → Backups folder
📱 Moving to a new device or new version:
  1. Download backup on your current device → save to Google Drive
  2. Open the new file → go to Live Calendar → ➕ Add event → scroll here
  3. Tap 📂 Choose backup file → select the JSON → all data restored ✅
🔒 Your data never leaves your device — backup files are private JSON files, not uploaded anywhere.
